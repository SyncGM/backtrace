- (void) update(*args, &block)

Note:

The original method did not take the possibility of a dead fiber into account; as such, calling @fiber.resume on a dead fiber after handling an exception would cause a new FiberError to be raised, eventually resulting in a stack overflow.

This method returns an undefined value.

def update(*args, &block)
  ses_backtrace_gi_upd(*args, &block)
rescue FiberError
  # The fiber is most likely dead due to exception handling -- increment the
  # index for the interpreter and create a new fiber to interpret event
  # commands.
  @index += 1
  create_fiber
end
